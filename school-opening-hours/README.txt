Work Schedule - Time Table.

== Description ==

Simple plugin for showing work schedule and message whether we work today or not.

Thanks WordPress.org and github.com

== Installation ==

1. Upload the entire `School Opening Hours` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
